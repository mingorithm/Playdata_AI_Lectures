import pandas as pd

컬럼 값 설명¶

- SeriousDlqin2yrs :target, 최근 2년동안 90일 이상 연체한 적 있는지 여부 (1: 양성 , 0: 음성 )
- RevolvingUtilizationOfUnsecuredLines
    - 부동산과 할부 부채를 제외한 보유 자산 및 신용 대비 현재 운용할 수 있는 돈의 비율
- age : 나이
- NumberOfTime30-59DaysPastDueNotWorse : 최근 2년동안 30일 ~ 59일 연체한 횟수
- DebtRatio : 전체수입 대비 월 부채 상환과 월 지출 합계 비율 
- MonthlyIncome : 월 수입
- NumberOfOpenCreditLinesAndLoans : 대출자가 보유중인 담보 대출 및 신용 대출 건수
- NumberOfTimes90DaysLate : 과거 90일 이상 연체한 횟수
- NumberRealEstateLoansOrLines : 주택 담보 대출 포함한 부동산 담보 대출 건수
- NumberOfTime60-89DaysPastDueNotWorse : 최근 2년간 60~89일 연체한 횟수
- NumberOfDependents : 대출자를 제외한 부양가족 수

대출 위험도 예측 모델 순서¶

- shape, column, describe, info, head 등 다양한 data 분석
- NA 값 처리, Outlier 처리, 정규화 ( Scaling ) , 범주형 확인, 실수 변환, feature 다루기
- Correlation에 대해 정리
- 전처리 후 모델 생성
    - 전처리로 score가 높은건지, 튜닝으로 score 가 높은건지 HyperParameter 값을 Default로 준 모델 생성
    - 전처리와 튜닝을 완벽히 분리하는 것이 중요
    - DecisionTree, LogisticRegression,Xgboost,RandomForest ( 배운 것에서 활용해보기 )
    - 이진 분류의 기본파라미터를 가진 데이터를 생성 후 적합한지 테스트 진행 ( Train, Val )
    - 기본파라미터로 과적합이 나올 경우 문제 정도는 해결해 줄 것 ( SVM 사용 X)
    - GridSearchCV활용
    - Down-Sampling에 대해 알아볼 것
    - Smote 에 대해 알아볼 것
- Test Set으로 최종 진행할 것
  • 나중에 metrics.py 파일 가져올 것

def fill_depen_value(data):
    if pd.isnull(data["NumberOfDependents"]) and data["cate_age"] in ["20대","60대","70대","80대","90대 이상"]:
        return 0
    elif pd.isnull(data["NumberOfDependents"]) and data["cate_age"] in ["30대","40대","50대"]:
        return 1
    return data["NumberOfDependents"]
data["NumberOfDependents"] = data.apply(fill_depen_value,axis=1)
